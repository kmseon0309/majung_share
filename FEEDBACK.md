# 프로젝트 피드백 (2026-06-15)

## 기능 개선

### 1. 첫 멘트 → 푸시 알림 연동
- 푸시 알림을 통해 진입하거나 그렇지 않더라도, **앱 첫 멘트 내용을 푸시 알림 메시지와 동일하게** 구성

### 2. AI 행동 추천 텍스트 간략화
- 행동 추천 답변은 **2줄 이하**로 제한
- **명사형**으로 마무리 (예: "~하기", "~산책")

### 3. 일기 삭제 시 행동 추천도 함께 삭제
- 일기를 삭제하면 연결된 **행동 추천 항목도 자동 삭제**

## UI 수정

### 4. 홈화면 행동 추천 카드 텍스트 잘림 개선
- 두 줄짜리 행동 추천 텍스트에서 **두 번째 줄 하단이 잘리는 문제** 수정
- 카드 높이 또는 overflow 처리 개선 필요

## 품질 검토

### 5. 존댓말 / 반말 철저 적용
- AI가 생성하는 **모든 텍스트**에 대해 사용자 설정(존댓말/반말)이 일관되게 적용되는지 확인
- 누락된 곳이 있으면 수정
