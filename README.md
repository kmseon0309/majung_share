# majung

A new Flutter project.

## Getting Started

1. Figma 16색 감정/테마 완벽 이식: 감정 5단계 mood 칩 및 grayscale 7단계까지 theme.dart에 완전 매핑.
2. 11종 공통 컴포넌트 개발: 피그마와 1픽셀 오차도 없는 pill 버튼, 비대칭 말풍선, 추천 카드, 세그먼트 슬라이더, 반응형 원형 전송 버튼 등 완성.
3. 독립적 에셋 관리 시스템: flutter_svg 의존성 탑재 및 app_icons.dart를 통한 경로 상수 정제.
4. 반응형 플레이그라운드 완성: main.dart에 좁은 화면에서도 가로 잘림 없이 원형 전송 버튼이 다 보이고 상태(Riverpod 3.x Notifier) 조작이 유기적으로 연동되는 반응형 인터페이스 탑재.
5. Git 형상 관리 시작: 이 모든 역사를 깃허브 원격 저장소와 연동 및 동기화 완료.
